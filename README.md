Libraries and frameworks to use:

Web3.js (https://github.com/ChainSafe/web3.js): A library to interact with Ethereum nodes
Ethers.js (https://github.com/ethers-io/ethers.js): An alternative to Web3.js with a smaller bundle size and additional features
Bootstrap (https://getbootstrap.com): A popular CSS framework for responsive and mobile-first frontend components
jQuery (https://jquery.com): A popular JavaScript library to simplify DOM manipulation
